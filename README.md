## 🧭 What this is

Control-Note is a concise lecture note on control theory for engineering use. It gives you a clear path through the core ideas used in robotics, automation, and feedback systems.

This project includes material in Chinese and English, so you can study in the language that fits you best. It is built for reading, review, and quick reference on Windows.
